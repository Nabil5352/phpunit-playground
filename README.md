# PHPUnit project

## Requirement

-   Any server ([Apache](https://httpd.apache.org/download.cgi) or [Nginx](http://nginx.org/en/download.html))
-   [php7+](https://www.php.net/manual/en/install.php)
-   [composer](https://getcomposer.org/)

## Installation

-   composer install

## Run

```
composer test
```
